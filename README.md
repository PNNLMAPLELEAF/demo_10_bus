# demo_10_bus
Ten Bus Demo Test Case

XML model needs to be uploaded using CIMHub

Demo app can be run after installing latest gridappsd-python
`pip install gridappsd-python --upgrade`

Run demo with
`python3 demo_app_runner.py`
